Colorbox panels is the most easiest drag & drop icon box and content box builder for WordPress. You can add unlimited panels with unlimited colour scheme.

== Description ==

Colorbox panels is based on bootstrap framework, so box is responsive and friendly to any mobile devices. Colorbox is integrated with masonry/isotope effect. You can add multiple content box and icon box on multiple pages and post. There unlimited colour scheme and font style available for content.

This is not a popup box or lightbox plugin, this  is a content box or icon box plugin, which is display your content in column layout. You can add service page or, team page using colorbox plugin. There Widget option also available, just paste your shortcode [WPSM_COLORBOX id=xx] in text widget for display your content in widget area.

== Installation ==

1. Upload the entire `colobox-panels` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Colorbox Panels menu to customize this plugin.
4. Copy your publish Colorbox shortcode and then paste into your page or post to display it.
